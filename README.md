# Registro MEI

## Descrição

O Registro MEI é um sistema desenvolvido em Java para gerenciamento de dados de Microempreendedores Individuais (MEI). O projeto permite realizar operações básicas de cadastro e manutenção de registros, facilitando o controle e a consulta das informações cadastradas.

## Funcionalidades

* Cadastrar novos registros de MEI;
* Consultar informações cadastradas;
* Alterar dados existentes;
* Excluir registros;
* Limpar os campos para um novo cadastro.

## Tecnologias Utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Banco de Dados Relacional
* JDBC

## Objetivo do Projeto

Este projeto foi desenvolvido com fins acadêmicos para aplicar conceitos de desenvolvimento Java, integração com banco de dados e operações CRUD (Create, Read, Update e Delete).

## Como Utilizar

1. Execute a aplicação.
2. Realize o cadastro de um novo MEI.
3. Consulte os registros cadastrados.
4. Atualize informações quando necessário.
5. Exclua registros que não forem mais necessários.
6. Utilize a opção "Novo" para limpar os campos e iniciar um novo cadastro.
